نوشتن این بات تقریبا وقت زیادی نگرفت (قبول دارم زیاد فرانت خوبی نداره و تو آینده شاید بهترش کردم)
این پروژه رو برای خودم نوشتم که کار های روزمرم رو بهتر مدیریت کنم

شما هم میتونید ازش استفاده کنید

<h3>
How it Work
</h3>
<em>
  <ul>
